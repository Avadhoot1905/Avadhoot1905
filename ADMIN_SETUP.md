## 🚀 Deployment Steps

### Step 1: Deploy Static Site to S3

```bash
# Build the static site
npm run build

# Deploy to S3
aws s3 sync out/ s3://your-bucket-name/ --delete

# Configure S3 for static hosting
aws s3 website s3://your-bucket-name/ \
  --index-document index.html \
  --error-document 404.html
```

### Step 2: Deploy Admin Lambda

```bash
cd lambda/admin

# Install dependencies
npm install

# Create deployment package
zip -r function.zip . -x "*.ts" "tsconfig.json" "*.md"

# Deploy to AWS Lambda
aws lambda create-function \
  --function-name portfolio-admin \
  --runtime nodejs20.x \
  --handler index.handler \
  --role arn:aws:iam::ACCOUNT_ID:role/lambda-role \
  --zip-file fileb://function.zip \
  --environment Variables="{ADMIN_SECRET=your-secret,DATABASE_URL=your-neon-url}"
```

### Step 3: Create API Gateway

```bash
# Create HTTP API
aws apigatewayv2 create-api \
  --name portfolio-admin-api \
  --protocol-type HTTP \
  --target arn:aws:lambda:REGION:ACCOUNT_ID:function:portfolio-admin

# Get the API endpoint URL
# It will be: https://API_ID.execute-api.REGION.amazonaws.com/admin/chats
```

### Step 4: Update Admin UI

1. Edit `public/admin.html` line 210
2. Replace:
   ```javascript
   const API_URL = 'https://YOUR_API_GATEWAY_URL/admin/chats';
   ```
3. Rebuild: `npm run build`
4. Redeploy: `aws s3 sync out/ s3://bucket/ --delete`

## 🔐 Environment Variables

Set these in AWS Lambda:

| Variable | Value | Where to Get |
|----------|-------|--------------|
| `ADMIN_SECRET` | `my-secure-password-123` | Create a strong password |
| `DATABASE_URL` | `postgresql://...` | Copy from Neon dashboard |

**⚠️ NEVER commit these to Git!**
